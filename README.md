## 📱 客管アプリについて

客管アプリは、接客業におけるお客様管理を効率化するためのモバイルアプリケーションです。

### 主な機能
- 顧客情報の登録・管理
- 来店履歴の記録
- タグ機能による顧客分類
- クラウドバックアップ機能（プレミアム）
- データの検索・フィルタリング
